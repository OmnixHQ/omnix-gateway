# Security Policy

## Supported Versions

Only the latest release of UCP Gateway is supported with security updates.

| Version  | Supported |
| -------- | --------- |
| Latest   | Yes       |
| < Latest | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly.

**Email:** [momentum.group139@gmail.com](mailto:momentum.group139@gmail.com)

Please include:

- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant logs or screenshots
- Your assessment of the severity

## Response SLA

- **Acknowledgment:** within 48 hours of receiving your report
- **Critical fix:** within 7 days of confirmed critical vulnerability

## Scope

This policy applies to all code in this repository.

## Disclosure

We ask that you do not publicly disclose the vulnerability until we have had a chance to address it. We will coordinate disclosure timing with you.
